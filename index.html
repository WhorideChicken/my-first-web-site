산타세요?

1. 상대방에게 익명으로 쪽지를 보낼 수 있음
2. 작성자는 쪽지 작성 후 기존 사진 or 즉석 셀카를 통해 어노잉 오렌지같은 캐릭터 이미지 생성
3. 쪽지 + 캐릭터 (커스텀 가능 → 우선 산타 한정)
4. 카메라 찍을 때 부터 산타 필터 생성 (스노우 마냥)
5. 상자 모양과 디자인 선택 후 상대방에게 보내기
6. 상대방의 화면은 점점 쌓여가는 선물상자 보여주기
7. 쪽지 선물 받은 사람은 인스타 공유해서 산타 뜨끔하게 하기(이 어노잉산타 누구셈~)
